Bibliografia del personal

Logros

De los primero logros fue el sistema de ventanas llamado SunDew, luego renombrado NeWS, , el Sistema de Windows extensible en red. En junio de 1991, Gosling, Mike Sheridan, Patrick Naughton y Ed Frank iniciaron el proyecto de lenguaje Java (llamado “El Proyecto Verde” en ese momento.)

Desarrolló una variante en el popular editor de pantallas Emacs y escribió una versión del sistema operativo UNIX para sistemas informáticos multiprocesador.

Logro portar el código p Pascal de UCSD desde una estación de trabajo PERQ para ejecutarlo en un sistema informático DEC VAX escribiendo un emulador VAX.

Fue elegido miembro de la Academia Nacional de Ingeniería de Estados Unidos (NAE) y por un tiempo fue investigador de investigador de IBM.

Construyó un editor de texto WYSIWYG, un editor de dibujo basado en la restricción y un editor de texto llamado de Emacs para sistemas Unix, que se convirtió en el más utilizado editor de texto de Unix.

La World Wide Web le permitió conjurar un sistema donde "applets" de las solicitudes se mueven a través de Internet y proporcionar capacidades multimedia en cualquier equipo.

En febrero de 2007, fue nombrado como Oficial de la Orden de Canadá; es el más alto honor civil de ese país.

Estudios
